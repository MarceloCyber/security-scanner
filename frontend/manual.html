Introdução

Bem-vindo à Iron Net

A Iron Net é uma plataforma profissional de análise de segurança que oferece diversas ferramentas para identificar vulnerabilidades em sistemas, redes e aplicações.

Objetivo da Plataforma

Auxiliar profissionais de segurança, desenvolvedores e administradores de sistemas a identificar e corrigir vulnerabilidades antes que sejam exploradas por agentes maliciosos.

Para quem é esta plataforma?

  • Profissionais de Segurança da Informação
  • Pentesters e Ethical Hackers
  • Desenvolvedores de Software
  • Administradores de Sistemas e Redes
  • Estudantes de Cibersegurança

Ferramentas Disponíveis

Port Scanner

Gratuito

O que faz?

O Port Scanner identifica portas abertas em um servidor ou dispositivo de rede, detectando serviços em execução e possíveis vulnerabilidades de configuração.

Como usar?

1
Acesse a ferramenta

No dashboard, clique em "Port Scanner"

2
Insira o alvo

Digite o IP ou domínio que deseja escanear (ex: 192.168.1.1 ou example.com)

3
Configure as portas

Escolha o range de portas (ex: 1-1000 ou portas específicas: 80,443,3306)

4
Execute o scan

Clique em "Iniciar Scan" e aguarde os resultados

Informações fornecidas:

  • Portas abertas e fechadas
  • Serviços em execução
  • Versões de software detectadas
  • Banners dos serviços
  • Nível de risco de cada porta
Atenção

Apenas escaneie sistemas que você possui ou tem autorização explícita para testar. Scans não autorizados podem ser ilegais.

Code Scanner

Premium

O que faz?

Analisa código-fonte em busca de vulnerabilidades de segurança, más práticas e padrões de código inseguro em diversas linguagens de programação.

Como usar?

1
Selecione o arquivo

Faça upload do arquivo de código (.py, .js, .php, .java, etc.)

2
Aguarde a análise

O sistema irá processar o código automaticamente

3
Revise os resultados

Analise as vulnerabilidades encontradas por nível de severidade

4
Corrija as falhas

Implemente as correções sugeridas em seu código

Vulnerabilidades detectadas:

  • SQL Injection
  • Cross-Site Scripting (XSS)
  • Command Injection
  • Path Traversal
  • Hardcoded Credentials
  • Weak Cryptography
  • Insecure Deserialization
  • E muitas outras...
Dica

Execute o Code Scanner regularmente durante o desenvolvimento para identificar problemas antes de colocar o código em produção.

API Scanner

Premium

O que faz?

Testa APIs REST em busca de vulnerabilidades de autenticação, autorização, injeção e outras falhas de segurança específicas de APIs.

Como usar?

1
Insira a URL da API

Digite o endpoint completo da API (ex: https://api.example.com/v1)

2
Configure autenticação (se necessário)

Adicione tokens, API keys ou credenciais necessárias

3
Execute os testes

O scanner testará diversos vetores de ataque automaticamente

Testes realizados:

  • Autenticação e Autorização
  • Injeção de SQL/NoSQL
  • Rate Limiting
  • CORS Misconfiguration
  • Information Disclosure
  • Mass Assignment

Dependency Scanner

Premium

O que faz?

Analisa as dependências do seu projeto (bibliotecas, frameworks, módulos) e identifica versões com vulnerabilidades conhecidas (CVEs).

Como usar?

1
Faça upload do arquivo

Envie o arquivo de dependências (package.json, requirements.txt, pom.xml, etc.)

2
Análise automática

O sistema verifica cada dependência em bancos de dados de vulnerabilidades

3
Revise o relatório

Veja quais pacotes têm vulnerabilidades e versões seguras recomendadas

Informações fornecidas:

  • CVEs conhecidos
  • Severidade das vulnerabilidades
  • Versões seguras recomendadas
  • Dependências transitivas vulneráveis

Docker Scanner

Premium

O que faz?

Analisa imagens Docker em busca de vulnerabilidades, más configurações e problemas de segurança em containers.

Como usar?

1
Upload do Dockerfile ou imagem

Envie o Dockerfile ou especifique a imagem Docker

2
Análise de camadas

O scanner analisa cada camada da imagem

3
Resultados detalhados

Receba relatório com vulnerabilidades e recomendações

Verificações realizadas:

  • Vulnerabilidades em pacotes do sistema
  • Configurações inseguras
  • Secrets expostos
  • Permissões inadequadas
  • Imagens base desatualizadas

GraphQL Scanner

Premium

O que faz?

Testa APIs GraphQL em busca de vulnerabilidades específicas como introspection habilitada, queries excessivas, injection e falhas de autorização.

Como usar?

1
Insira o endpoint GraphQL

Digite a URL do endpoint (ex: https://api.example.com/graphql)

2
Configure headers (opcional)

Adicione tokens de autenticação se necessário

3
Inicie o scan

O sistema testará diversos vetores de ataque

Testes realizados:

  • Introspection habilitada
  • Query depth excessiva
  • Batch attacks
  • Field duplication
  • Injection vulnerabilities
  • Authorization bypass
