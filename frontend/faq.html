Geral

O que é o Security Scanner?

O Security Scanner é uma plataforma completa de análise de segurança que oferece diversas ferramentas para identificar vulnerabilidades em sistemas, redes, aplicações e código-fonte.

Nossa plataforma foi desenvolvida para profissionais de segurança, desenvolvedores e administradores de sistemas que desejam melhorar a segurança de suas aplicações e infraestrutura.

Preciso de conhecimento técnico para usar a plataforma?

O nível de conhecimento necessário varia de acordo com a ferramenta:

  • Port Scanner: Conhecimento básico de redes
  • Code Scanner: Conhecimento de programação
  • Outras ferramentas: Conhecimento intermediário a avançado em segurança

Oferecemos documentação completa e manual de uso para ajudá-lo a começar.

A plataforma é legal de usar?

Sim, a plataforma é 100% legal quando usada corretamente. No entanto, você DEVE:

  • Obter autorização por escrito antes de escanear qualquer sistema
  • Usar apenas em sistemas que você possui ou tem permissão para testar
  • Seguir todas as leis locais e internacionais sobre testes de segurança

Importante: Scans não autorizados são ilegais e podem resultar em consequências legais graves.

Planos e Pagamentos

Quais são os planos disponíveis?

Oferecemos 4 planos:

  • FREE (R$ 0/mês): 10 scans/mês, apenas Port Scanner
  • Starter (R$ 49/mês): 100 scans/mês, todas as ferramentas
  • Professional (R$ 149/mês): Scans ilimitados, API access, suporte prioritário
  • Enterprise (R$ 499/mês): Tudo do Professional + recursos avançados

Como funciona o limite de scans?

Cada scan realizado (port scan, code scan, etc.) conta para o seu limite mensal. O limite é renovado automaticamente no início de cada mês.

Planos Professional e Enterprise têm scans ilimitados.

Posso fazer upgrade do meu plano a qualquer momento?

Sim! Você pode fazer upgrade do seu plano a qualquer momento através da página de Pricing no dashboard. O valor será calculado proporcionalmente ao tempo restante do mês.

Quais formas de pagamento são aceitas?

Aceitamos:

  • Cartão de crédito (Visa, Mastercard, American Express)
  • PIX (em breve)
  • Boleto bancário (em breve)

Todos os pagamentos são processados de forma segura através do Stripe.

Posso cancelar minha assinatura?

Sim, você pode cancelar sua assinatura a qualquer momento através do dashboard. Você continuará tendo acesso aos recursos do plano até o final do período já pago.

Não há multas ou taxas de cancelamento.

Ferramentas e Uso

Quanto tempo leva um scan de portas?

O tempo varia de acordo com o range de portas:

  • Portas comuns (1-1000): 30 segundos a 2 minutos
  • Range completo (1-65535): 10 a 30 minutos

Recomendamos começar com ranges menores para testes mais rápidos.

Quais linguagens o Code Scanner suporta?

O Code Scanner suporta as seguintes linguagens:

  • Python (.py)
  • JavaScript (.js)
  • PHP (.php)
  • Java (.java)
  • C/C++ (.c, .cpp)
  • Ruby (.rb)
  • Go (.go)

Os resultados dos scans são salvos?

Sim! Todos os scans realizados são salvos automaticamente e podem ser acessados na seção "Reports" do dashboard.

Planos Professional e Enterprise permitem exportar relatórios em PDF.

Posso escanear múltiplos alvos ao mesmo tempo?

Atualmente, você precisa realizar um scan por vez. No entanto, planos Enterprise têm suporte a scans em lote através da API.

Como interpretar os resultados do scan?

Os resultados incluem:

  • Severidade: Critical, High, Medium, Low - indica a gravidade da vulnerabilidade
  • Descrição: Explica o problema encontrado
  • Recomendação: Como corrigir a vulnerabilidade
  • Referências: Links para mais informações (CVEs, CWEs)

Priorize corrigir vulnerabilidades Critical e High primeiro.

Técnico e API

A plataforma possui API?

Sim! Planos Professional e Enterprise têm acesso completo à nossa API REST.

A API permite integrar o Security Scanner em seus sistemas, CI/CD pipelines e automações.

Consulte a documentação técnica para mais detalhes.

Existe limite de requisições na API?

Sim, temos rate limiting baseado no plano:

  • FREE: 10 requisições/minuto
  • Starter: 50 requisições/minuto
  • Professional: 100 requisições/minuto
  • Enterprise: 500 requisições/minuto

Como integrar com CI/CD?

Oferecemos exemplos de integração para:

  • GitHub Actions
  • GitLab CI
  • Jenkins
  • CircleCI

Consulte a seção de integrações na documentação.

Os dados são criptografados?

Sim! Todos os dados são criptografados:

  • Em trânsito: TLS/SSL (HTTPS)
  • Em repouso: Criptografia AES-256
  • Senhas: Hash bcrypt com salt

Levamos a segurança dos seus dados muito a sério.
